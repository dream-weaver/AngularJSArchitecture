# AngularJSArchitecture
This is large scale application architecture of AngularJS

## Features
- AngularJS 1.x
- RequireJS
- UI-Router
- OcLazyLoading
- RestAngular
- Satellizer

## Installation
```
git clone https://github.com/mdshohelrana/AngularJSArchitecture.git
cd AngularJSArchitecture
npm install
bower install
```
